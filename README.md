## MATOT

基于MAA全新架构的未定助手。图像技术 + 模拟控制，解放双手！
由[MaaFramework](https://github.com/MaaXYZ/MaaFramework)强力驱动！


## 可能会有的功能

- [x] 启动游戏：目前支持官服和 B 服。
- [x] 领取邮件
- [x] 领取友谊徽章
- [x] 领取基地奖励
- [x] 领取每日任务奖励
- [ ] 关卡辩论：目前可以复盘会展顾问和业务考核。
- [ ] 外勤委托：需要提前配好卡组，战力不足时会直接退出。
- [ ] 思绪残影：目前可以复盘，但要求刷取的残影至少有一个。
- [ ] 专属甜心制作：目前会自动制作第一件物品。

## 图形化界面
