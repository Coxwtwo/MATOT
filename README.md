## MATOT

基于MAA全新架构的未定助手。图像技术 + 模拟控制，解放双手！
由[MaaFramework](https://github.com/MaaXYZ/MaaFramework)强力驱动！


## 可能会有的功能

- [x] 启动游戏：目前支持官服和 B 服。
- [x] 领取奖励：目前包括邮件、友谊徽章、基地素材和每日任务。
- [x] 专属甜心制作：目前会自动制作第一件物品。
- [x] 辩论：目前支持一键复盘。
- [x] 关卡：目前支持会展顾问和业务考核。
- [ ] 思绪残影
